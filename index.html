    Diskripsi Singkat

    Match Picture Game merupakan game yang yang cara mainnya mencocokan gambar antara gambar satu dengan gambar yang lainnya. Selain itu, pada game ini juga ada fitur permainan dimana cara mainnya dengan menebak nama gambar tari yang telah disediakan. Tujuan dari dibuatnya game ini sebagai media pembelajaran agar user semakin mencintai kebudayaan Indonesia. Dalam Game ini berisi berbagai tarian tradisional Indonesia. User ditantang untuk mengasah kemampuannya tentang pengetahuan kebudayaan Indonesia terutama tentang tarian tradisional di Indonesia. User juga ditantang untuk mengasah ingatannya dengan mencocokkan gambar tarian yang disediakan.

    Cara Memainkan

    Dalam game ini disediakan 2 model permainan

    1. Tebak Gambar

      2. Tebak nama Gambar di kolom sebelah kiri dengan mengisi textbox yang berada di sebelah kanan.

      • klik tombol OK untuk mengecek kebenaran tebakan
      • Untuk kembali ke menu utama klik tombol "MENU"
      • Untuk melihat cara main klik tombol "HELP"
    2. Cocokan Gambar

      3. Dalam Game mencocokkan gambar ini mengadu ketangkasan dan ketelitian User. Ingatan yang bagus juga sangat diperlukan, karena user harus mengingat letak pasangan gambar pada window yang telah tersedia.

      • Silahkan klik 2 kolom gambar
      • Pastikan 2 kolom gambar tersebut memiliki gambar yang sama
      • Apabila sudah terbuka semua gambar, maka Anda telah berhasil menyelesaikan game cocokkan gambar
      • Untuk kembali ke menu utama klik tombol "MENU"
      • Untuk melihat cara main klik tombol "HELP"
